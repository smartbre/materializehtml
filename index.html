Layout & Spacing


Almost all the content of this website was coded to be displayed inside of a container. I decided to place all information into containers and then pertinent information inside of cards because to help with readibility.

Navigation Bar

The navigation bar is seperate from the page's content. The navigation bar spanned the entire page on websites with great design. When the navigation bar is contained inside the container, it did not take up the entire page.

Cards

Initially the information was contained inside of uniform-sized small cards that wasn't aesthetically pleasing. So, I assigned the class card-panels. Since I wanted to mention 3 aspects of great web design with regards to layout and spacing, I divided each column into 3 groups of 4 for a total of 12 column spaces.

Spacing

I inserted a line break after each heading to help space out material on the page.

Colors and Fonts


Color Scheme

This website demonstrates a color scheme that is complementary. The navigation bar is orange and most of the cards grey-blue. I also experimented with darkening each color and decided to go for a darker shade for both so that the white text could stand out.

Font Size

I experimented with different font sizes and ended up using "h1" for headings and "h4" for subheadings (inside the cards).
